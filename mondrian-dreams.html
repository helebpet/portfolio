Mondrian Dreams

2024

Generative Art, Interactive Web Experience

View Live Project

Concept

Mondrian Dreams transforms Piet Mondrian's iconic geometric compositions into dynamic, algorithmically generated patterns. The project explores how computational processes can reinterpret established artistic movements. Ten distinct pattern types range from organic ripples and cellular structures to geometric mazes and plasma effects, each driven by mathematical functions combining trigonometry, distance calculations, and noise algorithms.

Technical Approach

The application renders thousands of cells in real time, with each color determined by calculations factoring position, time, and pattern variables. The palette is strictly constrained to Mondrian's signature primaries of red, yellow, blue, black, and white. The responsive grid system adapts to any screen size while maintaining mathematical integrity.

Interaction

Users explore visual possibilities through simple clicks that cycle through pattern algorithms. Each interaction reveals a new mathematical interpretation of the Mondrian aesthetic. An export feature captures any moment as a high resolution PNG, allowing users to preserve and share compelling patterns. This bridges historical art movements with contemporary interactive design.
